# BirthMonth

This program asks the user for their birth month as an integer from 1 to 12.

If the number is in range, it prints:
`Your birth month is: N`

If the number is not in range, it prints:
`You entered an incorrect month value: N`

## Test Runs

Valid input:

```text
5
Your birth month is: 5
```

Invalid input:

```text
13
You entered an incorrect month value: 13
```
